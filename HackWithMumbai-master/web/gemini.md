# Credify Engineering Guidelines

Credify is a premium SaaS-grade proof-of-learning platform.

Tech Stack:
- Next.js
 15
- React
- TypeScript
- TailwindCSS
- shadcn/ui
- Framer Motion
- Firebase Authentication
- Firestore
- Firebase Storage

Design Philosophy:
- premium UI
- minimal modern aesthetic
- Stripe/Linear/Vercel inspired
- modular architecture
- scalable codebase
- reusable components
- production-quality code

Code Expectations:
- avoid monolithic files
- create reusable components
- use clean TypeScript interfaces
- use App Router best practices
- separate UI/business logic
- maintain clean folder organization
- use server/client components correctly
- prefer composable architecture

UI Expectations:
- dark futuristic aesthetic
- smooth animations
- subtle gradients
- responsive layouts
- loading skeletons
- polished dashboard experience

Important:
- blockchain should feel invisible
- UX should feel Web2-first
- avoid crypto-heavy terminology
- focus on credibility and verification
